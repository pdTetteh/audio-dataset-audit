# Manifest schema

## Required columns
- `item_id`
- `path`
- `split`
- `label`

## Optional but strongly recommended
- `speaker_id`
- `device_id`
- `collector_id`
- `date`
- `location`
- `language`
- `duration`
- `sample_rate`
- `channels`
- `source_dataset`

A richer manifest enables stronger checks, especially leakage analysis.
