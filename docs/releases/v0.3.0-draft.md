## AudioDatasetAudit v0.3.0

This release expands AudioDatasetAudit from metadata auditing into file-level dataset validation.

### Added
- audio probing with `soundfile`
- file existence checks
- audio readability checks
- sample-rate consistency checks
- channel consistency checks
- duration sanity checks
- manifest-vs-probed duration mismatch checks
- example probe manifest and report assets
- README screenshot and release assets

### Why this matters
A dataset can have clean metadata and still break experiments because files are missing, corrupted, inconsistent, or mislabeled in subtle ways. This release catches those issues before training or benchmark release.

### Example usage
```bash
python3 -m audiodatasetaudit.cli examples/probe_manifest.csv --format html --output examples/probe_report.html
```
